# graduate-management-system

## 项目描述
这是一个基于Web的信息学院毕业生就业数据管理系统，提供数据管理、查询和统计分析功能。

## 功能特点
- 毕业生信息增删改查
- 批量数据导入
- 就业数据统计分析
- 可视化图表展示

## 使用说明
1. 直接打开 `index.html` 即可使用
2. 数据存储在浏览器本地存储中

## 技术栈
- HTML5, CSS3, JavaScript
- Bootstrap 5.3 前端框架
- Chart.js 图表库
